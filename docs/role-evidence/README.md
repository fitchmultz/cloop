# Optional Demo and Enablement Materials

This folder contains optional demo, workshop, and enablement materials.

These files can be useful if someone wants to use the project in a walkthrough, workshop, or solution-design conversation, but they are not required to understand, run, or trust the repository.

- `evidence-map.md`
- `demo-script.md`
- `workshop-outline.md`
- `cookbook-guide.md`
- `operational-readiness-checklist.md`

For the primary public/reviewer path, use:
- `docs/reviewer_guide.md`
- `docs/architecture.md`
- `docs/reviewer_validation_checklist.md`
- `docs/release_readiness_report.md`

Use these optional materials alongside:
- `docs/release_readiness_report.md`
- `docs/reviewer_validation_checklist.md`
- `docs/public_release_checklist.md`
